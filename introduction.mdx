## What Makes It Special

### Accuracy Meets Performance

LemmingsJS-MIDI faithfully recreates the original Lemmings mechanics while achieving unprecedented performance. The engine can handle over 25 million simultaneous lemmings and run at speeds exceeding 60x the original game.

<Info>
  The engine includes steel terrain accuracy improvements and supports multiple Lemmings versions including Oh No! More Lemmings and various holiday editions.
</Info>

### Creative Tools

The built-in level editor provides professional-grade features:

- **Multi-select** with Shift-click or marquee selection
- **Clone objects** with Alt-drag
- **Full undo/redo** support
- **Level validation** to catch missing requirements
- **Import/export** for both .nxlv and classic .lvl formats

### MIDI Integration

Transform gameplay into music with the WebMIDI sequencer:

- Route game events to MIDI notes and chords
- Map lemming positions to velocity and timbre
- Configure ADSR envelopes per sound effect
- Control game speed and actions via MIDI input
- Real-time BPM syncing with configurable time signatures

### AI-Ready Architecture

The MCP (Model Context Protocol) server enables AI agents to:

- Inspect full game state via JSON
- Control lemmings and assign skills
- Capture vision via Playwright sessions
- Watch and assist in spectator mode
- Edit and test levels programmatically

## Version Coverage

LemmingsJS-MIDI supports multiple official Lemmings releases:

- **Lemmings** (original)
- **Oh no! More Lemmings**
- **Xmas 1991**
- **Xmas 1992**
- **Holiday 1993**
- **Holiday 1994**

Each version includes authentic levels, graphics, and mechanics.
