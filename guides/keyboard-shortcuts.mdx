## Custom Keybindings

### Configuration File

Keybindings are defined in `keybindings.json` at the project root:

```json keybindings.json
{
  "version": 1,
  "bindings": {
    "togglePause": ["Space"],
    "stepForward": ["BracketRight"],
    "stepBackward": ["BracketLeft", "Alt+BracketRight"],
    "selectSkillBuilder": ["KeyQ"]
  }
}
```

### Binding Syntax

<Steps>
  <Step title="Physical Key Codes">
    Use `KeyboardEvent.code` values (e.g., `KeyA`, `Digit1`, `Space`, `ArrowLeft`).
  </Step>
  
  <Step title="Modifiers">
    Join modifiers with `+`: `Shift+KeyT`, `Ctrl+KeyC`, `Alt+BracketRight`.
    
    Supported modifiers (case-insensitive):
    - `Ctrl` / `Control`
    - `Alt`
    - `Shift`
    - `Meta` / `Cmd` / `Command` / `Win`
  </Step>
  
  <Step title="Pure Modifiers">
    For modifier-only keys, use specific codes: `ShiftLeft`, `ShiftRight`, `ControlLeft`.
  </Step>
  
  <Step title="Multiple Bindings">
    Assign multiple key combinations to a single action using an array.
  </Step>
</Steps>

### Conflict Resolution

<Note>
  - Multiple actions can bind to the same chord; all dispatch in declaration order
  - Exact modifier matches take priority
  - Shift is optional unless Ctrl/Alt/Meta are pressed
  - Browser shortcuts (Ctrl/Alt/Meta combos) are preserved when no exact match exists
</Note>

### Example Customization

```json WASD Movement

  "panUp": ["KeyK"],
  "panRight": ["KeyL"]

```

```json Multiple Bindings
{
  "togglePause": ["Space", "KeyP", "Escape"],
  "restartLevel": ["Backspace", "KeyR"]
}
```
</CodeGroup>

## Design Notes

The keybindings system follows these principles:

1. **JSON-driven**: All actions are configurable without code changes
2. **Multiple bindings**: Each action supports multiple key combinations
3. **Stable defaults**: Built-in defaults remain when config is missing
4. **Runtime loading**: Config loads from `keybindings.json` via FileProvider
5. **Graceful fallback**: Invalid bindings are ignored without crashing

For implementation details, see [keybindings-design.md](https://github.com/doublemover/LemmingsJS-MIDI/blob/main/docs/keybindings-design.md).
